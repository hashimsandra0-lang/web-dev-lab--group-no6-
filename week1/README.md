# web-dev-lab--group-no6-
** practicals for web development course.
Objectives
General Objective
To design and develop a simple multi-page website using basic HTML concepts.

Specific Objectives
To understand the basic structure of an HTML document
To create multiple web pages (Home, About, Contact)
To implement internal links for navigation between pages
To add external links to other websites
To insert and display images (local and online)
To organize files using a proper folder structure
To use HTML comments for documentation

 What Was Done
Created a project folder (repository) with organized structure
Developed three web pages: index.html, about.html, and contact.html
Added a navigation menu to link all pages together
Inserted internal links between pages
Included external links (e.g., to other websites)
Added images:
Local images from the images folder
Online images using URLs
Used HTML comments to explain different parts of the code
Created a README.md file describing the project
Tested the website in a browser to ensure all links and images work correctly

Challenges Faced
Broken links due to incorrect file names or paths
Difficulty understanding the correct file structure
Images not displaying because of wrong folder paths
Confusion between internal and external links
Forgetting to close HTML tags properly
Case sensitivity issues (e.g., About.html vs about.html)
Initial difficulty understanding how pages connect together
Formatting issues when writing the README.md file
